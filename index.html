Environment design

How SciAgent Works

The agent receives a two-group dataset and a research question. It must form a hypothesis, select an appropriate statistical test, reason through the analysis, and draw a conclusion. A composite reward function evaluates every step.

STEP 01
📊
Observe Dataset
The agent receives two groups of measurements and a natural-language research question.
STEP 02
💡
Form Hypothesis
The agent states a testable hypothesis about the relationship between the groups.
STEP 03
🧪
Choose Statistical Test
The agent selects the appropriate test (Welch t-test for two independent groups with unknown variance).
STEP 04
✅
Draw Conclusion
The agent interprets results and states whether the difference is statistically significant.

Reward function

Composite Reward (0 → 1)

Each action is scored on five components — combining structural validity checks with ground-truth statistical correctness. Reward went from ~0.40 to 0.72+ after 300 GRPO steps.
